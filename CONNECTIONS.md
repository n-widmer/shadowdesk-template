## 2. Recommended for solo experts

If I haven't connected anything yet, these six are worth setting up first. Each one plugs in through the **claude.ai web Connectors panel** — open claude.ai in a browser → click my profile → **Settings → Connectors** → search the tool name → **Connect** → sign in with the underlying account. Once that handshake completes, every Claude Code session signed into my claude.ai account inherits the connection. No extra setup inside this folder.

| Tool | What it unlocks | How to connect |
|---|---|---|
| **Gmail** | Reading, drafting, and sending email on my behalf. | claude.ai → Settings → Connectors → search "Gmail" → Connect → sign in with Google. |
| **Google Calendar** | Reading my schedule, creating events, prepping for upcoming meetings. | claude.ai → Settings → Connectors → search "Google Calendar" → Connect → sign in with Google. |
| **Google Drive** | Reading docs, sheets, and slides I store in Drive. | claude.ai → Settings → Connectors → search "Google Drive" → Connect → sign in with Google. |
| **Notion** | Reading and writing pages and databases inside my Notion workspace. | claude.ai → Settings → Connectors → search "Notion" → Connect → sign in with Notion and pick the workspace. |
| **HubSpot** | Reading and updating contacts, companies, deals, and pipelines in my CRM. | claude.ai → Settings → Connectors → search "HubSpot" → Connect → sign in with HubSpot. |
| **Otter** | Pulling transcripts of my recorded meetings. | claude.ai → Settings → Connectors → search "Otter" → Connect → sign in with Otter. |

> On Microsoft 365 instead of Google? Outlook mail/calendar, OneDrive, and Teams live in the same Connectors panel — search the tool name and sign in with Microsoft.
